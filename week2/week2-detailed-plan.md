- else clause
- finally clause
- raise (creating exceptions)
- Custom exceptions

**Practice:**
```python
# Example: Safe division
def divide(x, y):
    try:
        result = x / y
    except ZeroDivisionError:
        print("Cannot divide by zero")
        return None
    else:
        return result
    finally:
        print("Division attempted")
```
